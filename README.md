# active-directory-lab
Built a Windows Server Active Directory environment to simulate real-world help desk and system administration taks including user management, group policies, and password resets
Objectives
  - Create Windows domain environment
  - Manage users and groups
  - Perform password resets and account unlocks
  - Apply Group Policy
  - Simulate help desk adminstrative taks
Technologies Used
  - Windows 11
  - Active Directory
  - Group Policy Management
Tasks Performed
  - Installed and configured Windows Server
  - Promoted server to Domain Controller
  - Created domain users and groups
  - Configured Organizational Units
  - Applied Group Policy
  - Tested user login and domain functionality
